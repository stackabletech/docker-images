The `hadoop-native.tar.gz` contains the Hadoop 3.2.2 pre-build native libraries as well as the https://rpmfind.net/linux/dag/redhat/el6/en/x86_64/dag/RPMS/snappy-1.0.5-1.el6.rf.x86_64.rpm[`libsnappy.so.1.1.3`] (AMD64).

Additionally, the JMX exporter config provides no rules to gather all metrics.

WARNING: This is for a demo to show Snappy compression and should not be used like that in our standard HBase packages!
