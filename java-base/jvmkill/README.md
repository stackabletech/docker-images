# jvmkill

Note: There are no upstream tags or releases, so we use the commit hash as release.

We are using https://github.com/airlift/jvmkill over https://github.com/cloudfoundry/jvmkill.

`cloudfoundry/jvmkill` is re-written in Rust, but hasn't seen a commit since 4 years.
Also, the official Trino docker image uses `airlift/jvmkill` as well.
